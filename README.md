# Weather App

## Description
A weather application built using HTML, CSS, and JavaScript that fetches real-time weather data using OpenWeather API.

## Features
- Search weather by city
- Temperature, humidity, wind speed
- Error handling

## Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## Agile Process
- Sprint 1: UI design
- Sprint 2: API integration
- Sprint 3: Error handling & testing
